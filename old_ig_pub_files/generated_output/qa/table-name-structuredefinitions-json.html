Open mHealth to FHIR Contained Patient Profile

The Open mHealth Contained Patient Profile defines the minimum expectations for the using a contained Patient resource within the Open mHealth DocumentReference profile to represent an opaque patient identifier.

Open mHealth to FHIR DocumentReference Profile

The Open mHealth (OMH) Document Reference Profile defines the minimum expectations for the using the DocumentReference resource to record, search and fetch omh schema in its native json format.

Open mHealth to FHIR Modality Extension

The modality whereby the measure is obtained. Choice of sensed|self-reported .

Open mHealth to FHIR Observation Profile

Open mHealth Observation Profile used as the target structure when mapping from OMH Schema
